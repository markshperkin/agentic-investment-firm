# web

React dashboard for the firm.

Screens: the live **event feed** (time-ordered trace of every agent/tool/guardrail action), the **approvals inbox** (Risk Committee — approve/edit/reject with the thesis card + citations), the **portfolio** (holdings, P&L vs SPY), and the **belief-evolution** view (per-ticker stance/confidence over the day).

See [`../docs/architecture.md`](../docs/architecture.md).
